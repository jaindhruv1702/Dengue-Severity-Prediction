# Dengue Severity Prediction using Machine Learning

## Overview

This project predicts dengue severity using machine learning techniques applied to clinical and laboratory patient data. The objective is to assist in early identification of severe dengue cases and support healthcare decision-making.

## Problem Statement

Dengue fever can progress from mild symptoms to severe complications. Early severity prediction can help healthcare professionals prioritize treatment and improve patient outcomes.

## Dataset

Dataset Source: Kaggle

The dataset contains patient clinical records and laboratory measurements associated with dengue severity classification.

### Data Preprocessing

## Results

The trained models achieved high predictive performance for dengue severity classification, with Random Forest providing strong feature importance insights.
